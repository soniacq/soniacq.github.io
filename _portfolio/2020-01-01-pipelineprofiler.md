PipelineProfiler is an interactive visualization tool for exploring and comparing the solution space of machine learning pipelines produced by AutoML systems.

It helps users inspect algorithm choices, pipeline structure, and performance tradeoffs in a compact visual form.

![PipelineProfiler teaser]({{site.url}}/images/vis_portfolio/PipelineProfiler_project.png)

**Focus**: AutoML interpretation, model comparison, and visual analytics.
