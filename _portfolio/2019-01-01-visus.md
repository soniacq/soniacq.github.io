Visus is an interactive system designed to support the model building process and curation of machine learning data processing pipelines generated by AutoML systems.

It also supports interactive data augmentation and visual model selection.

![Visus teaser]({{site.url}}/images/vis_portfolio/visus_project.png)

**Focus**: model curation, data augmentation, and human-in-the-loop ML.
